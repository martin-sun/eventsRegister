# 00 - 项目概览 / Project Overview

## 项目名称
萨斯卡通羽毛球双打锦标赛报名系统
Saskatoon Badminton Doubles Championship Registration System

## 项目背景
面向加拿大萨斯卡通（Saskatoon）羽毛球爱好者的赛事报名网站，主要服务华人社区，同时面向所有本地社区开放，因此需要**中英文双语**支持。

## 赛事基本信息
| 项目 | 详情 |
|------|------|
| 比赛日期 | 2026年5月24日（周日）上午 9:30 |
| 比赛地点 | Riverside Badminton & Tennis Club |
| 报名年龄下限 | 35周岁整（以2026年5月24日为准） |
| 兼项规则 | 每位选手只允许报名**一个**项目，不允许兼项 |
| 面向人群 | 萨省所有羽毛球爱好者，不限于华人社区 |
| 报名费 | $30 / 人 |

## 比赛项目
| 组别 | 年龄要求 | 报名上限 |
|------|----------|----------|
| 双打100岁组 | 两位选手年龄之和 ≥ 100（年龄取整，如40.5岁按40岁计算） | 32对（64人） |
| 双打80岁组 | 两位选手年龄之和 ≥ 80（年龄取整，如40.5岁按40岁计算） | 32对（64人） |

## 技术架构

```
┌─────────────────────────────────────────┐
│           SvelteKit (Frontend)          │
│  ┌───────────┐  ┌────────┐  ┌────────┐ │
│  │  Pages/   │  │  i18n  │  │ Store  │ │
│  │  Routes   │  │ zh/en  │  │        │ │
│  └─────┬─────┘  └────────┘  └───┬────┘ │
│        │                        │       │
│        └────────┬───────────────┘       │
│                 │                        │
│        ┌────────▼────────┐              │
│        │ Supabase Client │              │
│        └────────┬────────┘              │
└─────────────────┼───────────────────────┘
                  │ (直接连接，无后端API)
┌─────────────────▼───────────────────────┐
│           Supabase (Backend)            │
│  ┌──────────┐ ┌─────┐ ┌─────────────┐  │
│  │ PostgreSQL│ │ Auth│ │   Storage   │  │
│  │ Database  │ │     │ │ (sponsors   │  │
│  │ + RLS     │ │     │ │  logos etc) │  │
│  └──────────┘ └─────┘ └─────────────┘  │
└─────────────────────────────────────────┘
```

## 核心原则
1. **无后端 API** — 全部通过 Supabase Client SDK 直接与数据库交互
2. **中英文双语** — 使用 i18n 方案，用户可随时切换语言
3. **移动端优先** — 响应式设计，确保手机端体验良好
4. **安全性** — 通过 Supabase RLS（Row Level Security）保障数据安全
