# 03 - 国际化方案 / Internationalization (i18n)

## 方案选择：Paraglide JS + SvelteKit

使用 [Paraglide JS](https://inlang.com/m/gerre34r/library-inlang-paraglideJs) — 专为 SvelteKit 设计的 i18n 方案，编译时生成类型安全的翻译函数，零运行时开销。

## 支持语言
| 语言 | 代码 | 默认 |
|------|------|------|
| 中文（简体） | `zh` | 是 |
| English | `en` | 否 |

默认中文是因为主要受众为华人社区，但英文可通过语言切换器随时切换。

## 配置

```json
// project.inlang/settings.json
{
  "$schema": "https://inlang.com/schema/project-settings",
  "sourceLanguageTag": "zh",
  "languageTags": ["zh", "en"],
  "modules": [
    "https://cdn.jsdelivr.net/npm/@inlang/message-lint-rule-empty-pattern@latest/dist/index.js",
    "https://cdn.jsdelivr.net/npm/@inlang/message-lint-rule-missing-translation@latest/dist/index.js",
    "https://cdn.jsdelivr.net/npm/@inlang/plugin-message-format@latest/dist/index.js",
    "https://cdn.jsdelivr.net/npm/@inlang/plugin-m-function-matcher@latest/dist/index.js"
  ],
  "plugin.inlang.messageFormat": {
    "pathPattern": "./messages/{languageTag}.json"
  }
}
```

## 语言切换组件

```svelte
<!-- src/lib/components/LanguageSwitcher.svelte -->
<script lang="ts">
  import { languageTag, setLanguageTag, availableLanguageTags } from '$lib/paraglide/runtime';

  const labels: Record<string, string> = {
    zh: '中文',
    en: 'EN'
  };
</script>

<div class="flex gap-2">
  {#each availableLanguageTags as lang}
    <button
      class="px-3 py-1 rounded text-sm font-medium transition-colors
        {languageTag() === lang ? 'bg-primary text-white' : 'bg-gray-200 text-gray-700 hover:bg-gray-300'}"
      on:click={() => setLanguageTag(lang)}
    >
      {labels[lang]}
    </button>
  {/each}
</div>
```

## URL 策略

使用 **路径前缀** 方案：
- `https://example.com/zh/register` — 中文
- `https://example.com/en/register` — 英文
- `https://example.com/register` — 默认中文

Paraglide SvelteKit 插件会自动处理路由前缀。

## 数据库双语字段

数据库中需要双语的字段采用 `_zh` / `_en` 后缀：
- `categories.name_zh` / `categories.name_en`
- `sponsors.name_zh` / `sponsors.name_en`
- `players.name_zh` / `players.name_en`
- `teams.team_name_zh` / `teams.team_name_en`

前端根据当前语言选择对应字段显示。
