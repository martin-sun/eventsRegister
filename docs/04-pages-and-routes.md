# 04 - 页面结构与路由 / Pages & Routes

## 路由总览

| 路由 | 页面 | 权限 | 说明 |
|------|------|------|------|
| `/` | 首页 | 公开 | 赛事介绍、倒计时、快速报名入口 |
| `/register` | 报名页 | 公开 | 填写报名表单 |
| `/teams` | 参赛队伍 | 公开 | 展示已报名队伍列表 |
| `/rules` | 赛制规则 | 公开 | 赛制、让分规则说明 |
| `/sponsors` | 赞助商 | 公开 | 赞助商展示与致谢 |
| `/admin` | 管理后台 | 需登录 | 管理面板入口 |
| `/admin/registrations` | 报名管理 | 需登录 | 审核报名、标记付款 |
| `/admin/sponsors` | 赞助商管理 | 需登录 | 管理赞助商信息 |

### 2. 报名页 `/register`

**表单流程：**
```
Step 1: 选择组别
  ┌─────────────────┐
  │ ○ 双打100岁组   │
  │ ○ 双打80岁组    │
  └─────────────────┘
          ↓
Step 2: 填写选手1信息（报名人）
  ┌─────────────────────────────┐
  │ 英文姓名*:  [___________]  │
  │ 中文姓名:   [___________]  │
  │ 性别*:      ○男  ○女       │
  │ 出生日期*:  [YYYY-MM-DD]   │
  │ 邮箱*:      [___________]  │
  │ 电话:       [___________]  │
  │ 微信号:     [___________]  │
  └─────────────────────────────┘
          ↓
Step 3: 填写选手2信息（搭档）
  ┌─────────────────────────────┐
  │ （同上字段）                │
  └─────────────────────────────┘
          ↓
Step 4: 确认信息
  ┌─────────────────────────────┐
  │ 组别: 双打100岁组           │
  │ 选手1: John (男, 52岁)      │
  │ 选手2: Mary (女, 48岁)      │
  │ 年龄之和: 100 ✅             │
  │ 类型: 混双                   │
  │ 报名费: $60（$30/人）        │
  │                              │
  │ [确认提交]  [返回修改]       │
  └─────────────────────────────┘
          ↓
Step 5: 提交成功
  ┌─────────────────────────────┐
  │ ✅ 报名成功！                │
  │ 请通过以下方式支付报名费：   │
  │ • e-Transfer: xxx@email.com │
  │ • 现金：联系组委会           │
  │ 报名编号: #2026-0012        │
  └─────────────────────────────┘
```

**前端校验规则：**
1. 英文姓名必填
2. 出生日期必填，计算年龄 ≥ 35
3. 两位选手年龄之和满足组别要求
4. 邮箱格式校验
5. 实时显示年龄计算结果和组合类型（男双/女双/混双）

### 4. 赛制规则 `/rules`

**内容分区：**
1. **比赛形式** — 3局2胜制，21分
2. **让分规则** — 表格展示四种情况
3. **赛程说明** — 正赛 + 安慰赛赛程图
4. **报名须知** — 年龄要求、费用、截止日期

**让分规则表格：**
| 对阵 | 让分 | 起始比分 | 先发球 |
|------|------|----------|--------|
| 男双 vs 混双 | 6分 | 混双 6:0 | 混双 |
| 混双 vs 女双 | 6分 | 女双 6:0 | 女双 |
| 男双 vs 女双 | 11分 | 女双 11:0 | 女双 |
| 同性别 | 0分 | 0:0 | 抽签 |

### 6. 管理后台 `/admin`

**认证方式：** Supabase Auth — 仅限管理员邮箱登录。

**功能模块：**

#### 报名管理 `/admin/registrations`
- 查看所有报名队伍（含完整联系方式）
- 修改报名状态：pending → confirmed / cancelled
- 标记付款状态：unpaid → paid
- 导出报名列表（CSV）
- 手动添加/编辑报名

#### 赞助商管理 `/admin/sponsors`
- 添加/编辑/删除赞助商
- 上传赞助商 logo（Supabase Storage）
- 修改赞助商级别和确认状态

#### 仪表盘 `/admin`
- 各组别报名统计
- 收入统计（报名费 + 赞助金额）
- 最近报名列表

### Footer
