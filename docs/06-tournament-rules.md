# 06 - 赛制与让分规则 / Tournament Rules

## 比赛形式

| 项目 | 规则 |
|------|------|
| 赛制 | 3局2胜制 |
| 每局分数 | 21分 |
| 得分制 | 每球得分制（Rally Point） |
| 正赛 | 单败淘汰 + 三四名决赛 |
| 安慰赛 | 正赛首轮负者进入安慰赛，同为单败淘汰 |

## 让分规则

| 对阵情况 | 让分 | 起始比分 | 先发球方 |
|----------|------|----------|----------|
| 男双 vs 混双 | 6分 | 混双 6 : 0 男双 | 混双 |
| 混双 vs 女双 | 6分 | 女双 6 : 0 混双 | 女双 |
| 男双 vs 女双 | 11分 | 女双 11 : 0 男双 | 女双 |
| 男双 vs 男双 | 0分 | 0 : 0 | 抽签 |
| 混双 vs 混双 | 0分 | 0 : 0 | 抽签 |
| 女双 vs 女双 | 0分 | 0 : 0 | 抽签 |

**让分逻辑：** 性别优势方让分给性别劣势方。男双 > 混双 > 女双。

```typescript
// src/lib/handicap.ts

type GenderType = 'mens' | 'womens' | 'mixed';

interface HandicapResult {
  team1StartScore: number;
  team2StartScore: number;
  firstServe: 'team1' | 'team2' | 'coin_toss';
}

/**
 * 计算两队之间的让分
 * team1 为主队，team2 为客队
 */
export function calculateHandicap(
  team1Type: GenderType,
  team2Type: GenderType
): HandicapResult {
  const strength: Record<GenderType, number> = {
    mens: 3,
    mixed: 2,
    womens: 1,
  };

  const diff = strength[team1Type] - strength[team2Type];

  if (diff === 0) {
    // 同性别，无让分
    return { team1StartScore: 0, team2StartScore: 0, firstServe: 'coin_toss' };
  }

  if (diff === 1) {
    // 差一级：让6分（如男双vs混双，混双vs女双）
    return { team1StartScore: 0, team2StartScore: 6, firstServe: 'team2' };
  }

  if (diff === -1) {
    return { team1StartScore: 6, team2StartScore: 0, firstServe: 'team1' };
  }

  if (diff === 2) {
    // 差两级：让11分（男双vs女双）
    return { team1StartScore: 0, team2StartScore: 11, firstServe: 'team2' };
  }

  // diff === -2
  return { team1StartScore: 11, team2StartScore: 0, firstServe: 'team1' };
}
```

## 赛程结构（每组 32 对满员时）

### 正赛（Main Draw）— 单败淘汰

```
第1轮（32进16）：16场比赛
第2轮（16进8）：8场比赛
第3轮（8进4）：4场比赛
半决赛（4进2）：2场比赛
三四名决赛：1场比赛
决赛：1场比赛
─────────────────────────
正赛合计：32场
```

### 安慰赛（Consolation）— 正赛首轮负者

```
第1轮（16进8）：8场比赛
第2轮（8进4）：4场比赛
半决赛（4进2）：2场比赛
三四名决赛：1场比赛
决赛：1场比赛
─────────────────────────
安慰赛合计：16场（注：原文写15场，实为16场含三四名）
```

### 总计
| 项目 | 100岁组 | 80岁组 | 合计 |
|------|---------|--------|------|
| 正赛 | 32场 | 32场 | 64场 |
| 安慰赛 | 16场 | 16场 | 32场 |
| **总计** | **48场** | **48场** | **96场** |

### 时间估算
- 每场比赛预估用时：30分钟
- 总时长：96场 × 30分钟 = 2,880分钟 ≈ 48小时
- 如果有多片场地并行，例如 6 片场地：48 / 6 = 8小时

### 羽毛球用量估算
- 假设每场比赛打 2.5 局
- 总局数：96 × 2.5 = 240 局
- 每局 1 个球 → 240 个羽毛球 ≈ 20 桶（每桶12个）
- 每桶 $30 → 羽毛球费用约 $600
