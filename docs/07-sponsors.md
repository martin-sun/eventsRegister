## 赞助商权益详情

### 所有赞助商均享有：
- 网站赞助商页面展示（logo + 链接）
- 比赛日现场 banner 展示
- 社交媒体感谢帖

### 冠名赞助额外权益：
- 赛事全称包含赞助商名称
- 网站首页最大 logo + 独立展示区
- 所有宣传材料、海报首位展示
- 为100岁组正赛冠亚季军颁奖
- 现场独立展位
- 赛事照片/视频中赞助商标识

### 钻石/白金/黄金额外权益：
- 对应组别颁奖权
- 钻石/黄金赞助商在比赛间隙有站台机会
- 较大的 logo 展示（按级别递减）

## 网站展示规则

### Logo 尺寸规范
| 级别 | Logo 最大宽度 | 展示区域 |
|------|-------------|---------|
| 冠名 | 300px | 首页 Hero 区 + 独立区块 + 赞助商页顶部 |
| 钻石 | 200px | 首页赞助区 + 赞助商页 |
| 白金 | 160px | 首页赞助区 + 赞助商页 |
| 黄金 | 120px | 赞助商页 |
| 友情 | 80px | 赞助商页 |
| 媒体 | 120px | 页脚 + 赞助商页 |

### 首页展示
1. Hero 区下方：冠名赞助商独立展示
2. 赞助商横幅：钻石 + 白金 logo 滚动/排列
3. 页脚：所有赞助商 logo 小图

### 赞助商页面
按级别从高到低排列，每个级别一个区块，confirmed 的赞助商才展示。

```

├── title/
│   └── lin-tang-realty.png
├── diamond/
│   ├── td-bank.png
│   └── baydo.png
├── platinum/
│   └── slga.png
├── gold/
├── friend/
│   └── rbc.png
└── media/
    └── overseas-new-life.png
```

管理员通过后台上传 logo，前端从 Storage 公开 URL 读取。
